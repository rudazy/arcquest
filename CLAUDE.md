# Arc Terminal

Arc ecosystem quest and XP platform. The command center for every 
protocol building on Arc blockchain.

## What we are building
- Quest platform where users complete tasks across Arc protocols
- Earn XP for onchain actions + social tasks
- Level up publicly (Lv1 → Lv5)
- Claim soulbound + tradeable NFTs at milestones
- Project directory — official and community sections
- Built on Arc testnet (Circle's stablecoin L1)

## Stack
- Next.js 14 App Router, TypeScript, Tailwind, shadcn/ui
- Framer Motion for animations
- Privy for wallet connect
- Supabase for database
- Viem for onchain reads
- Solidity contracts on Arc testnet

## Key rules
- Never display wallet addresses publicly
- Community projects always show amber warning banner
- No wallet required to browse homepage
- XP awarded server-side only — never trust client
